## Basic Linux Commands
`pwd`       - print working director
`cat`       - print out content of file
`echo`      - repeat back input
`touch`     - create file
`rm`        - removes file
`mkdir`     - create directory
`rm -rf`    - removes contents in given folder and removes folder
`ls`        - lists contents in current directory
`>`         - redirects output into somewhere
`>>`        - appends output to somewhere
`&&`        - chains commands
    Ex: `mkdir testFolder && cd testFolder`

## Git Commands
`git add _______`           --> adds given file to commit to git
`git commit -m "_______"`   --> commits changes to be ready to push
`git push`                  --> pushes changes to GitHub
